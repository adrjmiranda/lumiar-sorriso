# Clínica Lumiar Sorriso

Uma landing page premium para a Clínica Odontológica Lumiar Sorriso, focada em excelência, tecnologia e atendimento humanizado.

## ✨ Tecnologias

- **React 18** + **Vite 6**
- **TypeScript** (Tipagem estrita)
- **Tailwind CSS v4** (Configuração baseada em variáveis de tema)
- **Framer Motion** (Animações de alta fidelidade)
- **Lucide React** (Ícones modernos)
- **React Router Dom 7**

## 🎨 Design & Estética

O projeto segue uma estética "High-Fidelity" com:

- Glassmorphism em elementos de interface
- Micro-animações suaves para interatividade
- Paleta de cores premium (Navy & Gold)
- Responsividade mobile-first completa

## 🚀 Como Executar

1. Clone o repositório:

```bash
git clone https://github.com/adrjmiranda/lumiar-sorriso.git
```

2. Instale as dependências:

```bash
npm install
```

3. Inicie o servidor de desenvolvimento:

```bash
npm run dev
```

## 📦 Deploy

O projeto está configurado para deploy automático no GitHub Pages:

```bash
npm run deploy
```
